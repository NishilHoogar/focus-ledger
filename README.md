# Focus Ledger

Focus Ledger is a lightweight Chrome extension that helps you track your web browsing activity, showing active and passive time per website. It’s designed for personal productivity and keeps all your data private and local.

## Features

- **Daily overview** of active vs passive time per website
- **Color-coded bars** for quick visual understanding
- **Fully offline** — no internet required for operation
- **Popup view** for daily stats
- **Optional dashboard** (coming soon) for weekly/monthly summaries and detailed analytics

## Privacy & Security

- All data is stored locally in your browser using `chrome.storage.local`.
- No external servers, APIs, or analytics are used.
- Your browsing information never leaves your computer.
- You can reset or export your data at any time (optional feature).

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/focus-ledger.git
   ```

2. Open Chrome and go to `chrome://extensions/`

3. Enable **Developer mode**

4. Click **Load unpacked** and select the project folder

5. Your extension should now be active
